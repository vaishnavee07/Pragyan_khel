═══════════════════════════════════════════════════════════════════════════
                             TECH STACK
═══════════════════════════════════════════════════════════════════════════

Backend:
  • Python 3.9+
  • FastAPI 0.109.0         (REST + WebSocket)
  • uvicorn 0.27.0          (ASGI server)
  • opencv-python 4.9.0.80  (Computer vision)
  • numpy 1.26.3            (Array operations)
  • psutil 5.9.8            (System monitoring)
  • websockets 12.0         (Real-time protocol)

Frontend:
  • HTML5                   (Structure)
  • CSS3                    (Styling, animations)
  • JavaScript ES6+         (Logic, WebSocket client)
  • WebSocket API           (Real-time communication)

Architecture:
  • Abstract Base Classes   (Polymorphism)
  • Service Layer Pattern   (Separation of concerns)
  • Strategy Pattern        (Switchable AI modes)
  • Observer Pattern        (Alert monitoring)
  • Dependency Injection    (Loose coupling)
